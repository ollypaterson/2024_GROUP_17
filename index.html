Group 17: VTK-based 3D Rendering & VR App

This repository houses our QT-based GUI applications designed for loading, visualizing and interacting with STL models under both desktop and VR environments. This was implemented using VTK and OpenVR.

How to Build and Run

  1. Clone this repo: 
    git clone https://github.com/ollypaterson/2024_GROUP_17.git
  2. Install dependencies: Qt, VTK, OpenVR
  3. Create a build folder and run CMake: 
    mkdir build && cd build  
cmake ..  
cmake --build . --config Release
  4. Run the application: 
    .\Release\YourApp.exe
